PUMPED Mobile App

AI-Powered Breast Milk Management System

PUMPED is a comprehensive iOS mobile application that revolutionizes breast milk management for new mothers through intelligent automation, AI-powered insights, and seamless hardware integration. Built with Swift and SwiftUI, the app connects to a custom IoT device to provide real-time monitoring, data analytics, and smart recommendations.

Key Features:

ü§ñ AI-Powered Assistant

ChatGPT-powered AI agent provides intelligent assistance for basic parenting tasks and questions. The AI assistant helps with general milk storage guidance, feeding advice, and answers common questions without accessing user data, ensuring complete privacy protection.

üî¨ Allergen Status Tracking

Simple allergen tracking system allows mothers to mark milk bags as allergen-free or containing allergens. The status is displayed in the inventory system for easy identification during feeding preparation and storage management.

üìß Smart Email Notifications

Nodemailer-powered notification system sends automated email alerts when milk inventory levels are running low. Users receive timely notifications to ensure adequate milk supply is maintained for feeding schedules.

üì± Native iOS Development

Built entirely in Swift with SwiftUI framework, delivering a smooth, responsive user experience with native iOS design patterns. Features include intuitive navigation, real-time data visualization, and seamless device integration.

PostgreSQL database backend ensures secure, scalable data storage for all milk measurements, user profiles, and historical analytics. Advanced data modeling supports complex queries and reporting capabilities.
