House Price Prediction ML Model

An interactive machine learning project that predicts house sale prices using a linear regression model. The model was trained on the Kaggle House Prices dataset after data preprocessing and a log-transformation of the target variable. The interactive web app was built using Python, Scikit-Learn, and Streamlit.

Users can input various house features and receive a predicted sale price instantly.
