Hardware Projects

Here are some of the hardware projects i've built

PCB Range Finder

Custom PCB created in Fusion 360, integrating both 3.3V and 5V regulators based on TI reference circuits. This project utilizes an MSP430 microcontroller paired with an ultrasonic sensor to measure distance. The MSP430 processes these signals, providing accurate readings while maintaining low power consumption. The compact PCB integrates power regulation, sensor connections, and headers for easy programming

PCB Range Finder Prototype

Here is the prototype for the PCB Range Finder. This shows the various electrical components and steps that went into creating the final product. This process included programming the MSP 430 using a breadboard to perform the tasks needed by the sensor on the PCB.
