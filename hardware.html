PCB Range Finder

Custom PCB created in Fusion 360, integrating both 3.3V and 5V regulators based on TI reference circuits. This project utilizes an MSP430 microcontroller paired with an ultrasonic sensor to measure distance. The MSP430 processes these signals, providing accurate readings while maintaining low power consumption. The compact PCB integrates power regulation, sensor connections, and headers for easy programming.

PCB Range Finder Prototype

Breadboard prototype for the PCB Range Finder showing the development process and component integration. This prototype phase included programming the MSP430 microcontroller to interface with the ultrasonic sensor, validate timing requirements, and test power consumption characteristics before PCB fabrication. Demonstrates iterative hardware development methodology.

Embedded Firmware Development

Comprehensive embedded firmware portfolio featuring real-time control systems, sensor integration, and communication protocols. The codebase includes MSP430 embedded C programming for the PCB range finder, ESP32 multi-threaded applications for sensor networks, and advanced control algorithms for automated systems. Features include interrupt handling, state machines, communication protocol implementation (SPI, I¬≤C, UART), and power management optimization.
